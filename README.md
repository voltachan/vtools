# 威科姆（VCOM）系计算机写保护配置工具
# 使用指南

## 1. 双击打开程序，直接点击“登录”

![截图](https://s2.ax1x.com/2019/02/23/khvarj.png)

## 2. 选择“关闭写保护”

![截图](https://s2.ax1x.com/2019/02/23/khvJG8.png)

## 3. 点击“确定”，系统自动重启。重启后，写保护消失。
